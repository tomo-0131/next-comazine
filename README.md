## COMAZINE

社内ブログをモダンなUIで公開・閲覧できるSaaSプロダクトです。

# プロダクト名の由来
- Company + Magazine = COMAZINE

# 使用技術
## Frontend
- Next.js(React.js)
- TypeScript(JavaScript)
- Tailwind CSS
- Headless UI, Chakra UI
- heroicons
- Emotion
- Moment
- Recoil

## Backend
- Firebase (version 9.x)
  Authentication, Firestore, Storage

## Hosting
